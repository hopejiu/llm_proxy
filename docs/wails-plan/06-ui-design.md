# UI 设计风格

## 整体风格

延续现有紫色主题风格，升级为更现代的桌面应用体验。左侧固定侧边栏导航 + 右侧内容区布局，替代原有的顶部导航栏，更符合桌面应用交互习惯。

## 整体布局

## 侧边栏

- 深色渐变背景（#1F2937 → #111827）
- Logo 区域含闪电图标 + "LLM Proxy" 文字
- 四个导航项垂直排列，每项含图标 + 文字
- 选中态：紫色高亮背景 + 左侧 3px 紫色指示条
- 底部固定区域显示代理服务状态（绿色圆点 + 端口号）
- 导航项：
  - 配置管理（齿轮图标）
  - 实时监控（闪电图标）
  - 统计报表（图表图标）
  - 运行日志（终端图标）

## 配置管理页 (Providers)

- 顶部操作栏：新增按钮 + 导入/导出 + CodeBuddy 配置
- Provider 卡片网格布局，每卡片含名称、URL、模型标签、别名标签、操作按钮
- 新增/编辑弹窗表单（页面特有模态框，包含在 providers.html 中）
- 删除确认弹窗（页面特有模态框）
- 复用现有 index.html + app.js 的 HTML 结构和逻辑

## 实时监控页 (Realtime)

- 顶部统计卡片行：活跃请求数、流式数、输出速率、刷新间隔
- 活跃请求列表（实时刷新，支持 Events 推送 + 低频轮询兜底）
- 最近完成的请求日志表格
- 活跃请求详情弹窗（共享模态框，在 index.html 中）
- 日志详情弹窗（共享模态框，在 index.html 中）
- 复用现有 realtime.html + realtime.js 的 HTML 结构和逻辑

## 统计报表页 (Stats)

- 顶部统计卡片行：今日/本周/累计 Token 和请求数
- 分时统计柱状图（ECharts，支持日期切换）
- 每日趋势折线图（ECharts，支持 Token/请求维度切换）
- 每日明细表格
- 最近请求日志表格 + 日志详情
- 日志详情弹窗（共享模态框，在 index.html 中）
- 复用现有 stats.html + stats.js 的 HTML 结构和逻辑

## 运行日志页 (Logs) - 新增

- 终端风格日志控制台，深色背景（#1F2937）
- 顶部工具栏：
  - 级别过滤按钮组（All/Debug/Info/Warn/Error）
  - 关键词搜索输入框
  - 暂停/继续按钮
  - 清屏按钮
- 日志流自动滚动到底部，暂停时停止滚动
- 每条日志按级别着色：
  - Error 红色
  - Warn 黄色
  - Info 白色
  - Debug 灰色
- 显示时间戳 + 级别 + 消息
- 使用 Wails Events 监听实时日志流
- 页面初始化时加载 RingBuffer 历史日志

## 共享模态框

以下模态框在 index.html 主框架中定义，全局唯一，避免 DOM ID 冲突：

### 日志详情模态框 (logDetailModal)

- 被 realtime 页面和 stats 页面共用
- 显示请求元信息（时间、模型、Provider、耗时、Token/s、状态、Input/Output/Cached/Total）
- 显示 Request Body 和 Response Body（可复制）
- common.js 中的 `showLogDetail(id)` 函数操作此模态框

### 活跃请求详情模态框 (activeDetailModal)

- 被 realtime 页面使用
- 显示活跃请求元信息（请求ID、模型、Provider、协议、状态、耗时、客户端IP、开始时间）
- 显示工具调用信息
- 显示 Request Body 和实时 Response Content
- realtime.js 中的 `showActiveDetail(requestID)` 函数操作此模态框

## 桌面特有交互

### 右键菜单

- 侧边栏导航项支持右键菜单（暂不实现，预留扩展点）

### 键盘快捷键

- `Ctrl+R`：刷新当前页面数据
- `Ctrl+1/2/3/4`：切换到对应页面
- `Esc`：关闭当前打开的模态框

### 窗口标题栏

- 使用 Wails 默认标题栏（Frameless: false），简化实现
- 标题显示 "LLM Proxy" + 代理服务状态
